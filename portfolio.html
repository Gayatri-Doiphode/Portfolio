Medical Reminder System
IoT + Android — Physical hardware project
● IoT APK Available Android ESP32

A smart medication reminder system that bridges an Android app with real IoT hardware. Patients set their medicine name and alert time in the app; at the scheduled moment, the ESP32 microcontroller fires a buzzer and displays the medicine name on an OLED screen — a physical, unmissable alert that works even without looking at a phone.

1
Pair via Bluetooth
Open the app — it scans and connects to the ESP32 board over Bluetooth. A green "Connected" banner confirms the hardware link is active.
2
Enter medicine name
Type the medicine name (e.g. Paracetamol) in the text field. This will display on the physical OLED screen when the alarm fires.
3
Set reminder time
Use the TimePicker widget to select the exact hour and minute the reminder should trigger on the hardware device.
4
Save & transmit
Tap "Save Reminder". The app confirms with "Reminder Sent" and transmits the medicine name and time to the ESP32 over Bluetooth.
5
Physical alert fires
At the set time, the ESP32 activates the buzzer (audible alarm) and shows the medicine name on the OLED display — a physical, unmissable alert for the patient.
